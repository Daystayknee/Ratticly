# Implementation Plan

## Phase 1 — Foundation (MVP)

Deliver:

- Auth + base navigation shell
- Dashboard v1 (core counts + alerts)
- Rat roster + add/edit rat
- Rat detail with overview, health, weight, timeline
- Relationship mapping (parents/offspring/siblings)

Exit criteria:

- Complete life record for an individual rat
- Daily breeder workflow possible without spreadsheets

## Phase 2 — Breeding Engine

Deliver:

- Pairing planner
- Pregnancy tracker + due alerts
- Litter management with per-pup milestones
- Basic lineage tree visualization

Exit criteria:

- End-to-end breeding cycle tracked from pairing to weaning

## Phase 3 — Operations

Deliver:

- Housing/cage boards
- Inventory + low-stock + expiration alerts
- Task center/reminder automation
- Reservations/placements/adopter management
- Finance ledger and summaries

Exit criteria:

- Operational overhead (supplies, tasks, placements, costs) centralized

## Phase 4 — Intelligence + Polish

Deliver:

- Advanced genetics prediction helpers
- Deep analytics and line health trends
- Memorial/retirement storytelling pages
- Export tools (pedigree sheets, CSV/JSON backups)
- Full premium theming suite

Exit criteria:

- Strategic breeding insights and premium presentation quality

## Non-Functional Requirements

- Role-based access controls
- Immutable audit events for critical changes
- Photo-safe backup/restore flows
- Responsive UX (desktop-first, tablet compatible)
- Query performance targets for roster and lineage views
