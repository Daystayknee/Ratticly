# Ratticly Product Spec

## 1) Dashboard

At-a-glance cards and widgets:

- Total rats, bucks, does, pups
- Active breeders, retired, available, reserved
- Pregnant does, litters in nest, litters weaned
- Observation/medical hold, urgent health flags
- Upcoming pairings, expected due dates, daily task list
- Supply low-stock alerts + cleaning/feed reminders
- Age milestones, birthdays, breeding/retirement readiness
- Financial snapshot (income, expenses, vet spend, margin)

Widgets:

- Quick stats cards
- Timeline (events due soon)
- Pregnancy progress ring
- Recent activity feed
- Low stock + health alert panels
- Birthdays/milestones module
- Breeding program snapshot
- Quick Add actions

## 2) Rat Roster

View modes:

- Photo card grid
- Dense table
- Compact breeder board
- Status board

Card fields:

- Photo, name, registered/call name, sex, age, DOB
- Variety + coat + markings
- Current status + breeding eligibility
- Cage/location, latest weight, health badge
- Availability and lineage snippet

Filters/search:

- Phenotype filters (coat, ears, markings, color, eyes)
- Lifecycle/state filters (pregnant, nursing, holdback, retired, deceased)
- Breeding/health filters (eligibility, flag, carrier traits)
- Search by name, ID, litter code, lineage notes

## 3) Rat Profiles

Sub-tabs:

- Overview
- Genetics
- Health
- Weight
- Breeding
- Relationships
- Litters
- Temperament
- Photos
- Notes
- Placement
- Timeline

Each profile supports:

- Identity + acquisition metadata
- Detailed phenotype + distinguishing marks
- Breeding quality + carrier/suspected traits
- Relationship graph links (family + social/cagemate)
- Complete event timeline across life cycle

## 4) Genetics & Trait Tracking

Taxonomy support:

- Coat/fur types
- Ear types
- Marking/pattern classes
- Color families and custom line notes
- Eye colors and special conditions

Tools:

- Genotype + phenotype fields
- Carrier trait matrix
- Pairing prediction helper
- Risk/compatibility warnings
- Trait probability outputs

## 5) Pairings + Pregnancy + Litters

Pairings:

- Buck/doe planner with candidate comparisons
- Compatibility score (health, temperament, relatedness)
- Goal traits + expected outcomes
- Status lifecycle (idea → review → approved → bred → complete/cancel)

Pregnancy tracking:

- Mating/conception estimates
- Due-date countdown
- Observation logs + alerts
- Nesting and postpartum checklists

Litter tracking:

- Litter metadata + survival stats
- Per-pup development logs and milestones
- Weaning/placement readiness
- Auto-creation of offspring records

## 6) Operations Modules

Health:

- Episodes, diagnostics, medications, follow-ups, vet costs

Housing:

- Cage inventory, occupancy, cleaning schedule, quarantine/maternity designations

Inventory:

- Supplies, medications, reorder points, expirations, supplier metadata

Placements:

- Waitlist, reservations, adopter records, contract state, assignment tracking

Finance:

- Income/expense ledger, litter profitability, cost trend reporting

Tasks:

- Unified reminder center for husbandry + breeding + placement workflows

## 7) Analytics + Memorials

Reports:

- Litter outcomes and survival
- Productivity by sire/doe/line
- Health trends by lineage
- Temperament and placement metrics
- Cost and inventory consumption trends

Memorials/retirement:

- Legacy and lineage impact for deceased rats
- Retirement records and comfort-care notes

## 8) UI/UX Requirements

- Soft pastel ambient background under clean white cards
- Delicate iconography and typography hierarchy
- Status chips and trait badges with clear color semantics
- Mobile-adaptive dashboard cards
- High-signal quick-add actions globally accessible
